Location Revenue Per Pound Runner

Instructions

To use this web application, follow the following steps:

  1. Run the Revenue Per Pound Report from Alsco CRS
    1. Go the Alsco CRS and log in
    2. Click on the 'Financial' category
    3. Click on Revenue Per Pound Report
    4. Enter the desired month, region, and plant
    5. Set the output type to xls
    6. Click the Run report button
    7. When the report is finished, click 'View Report' and download the 'Location Summary' file
    8. Choose that report as the first file on this site
  2. Run the National Account Listing report from Abel.
    1. Login to the branch Abel system.
    2. Run and save the report located under Processing > Reports > On Demand Reports > National Account Listing
    3. Open the csv file that was generated
    4. Save the file as an xlsx or xls file using Excel
    5. Choose that report as the second file on this site
  3. Click Submit and wait for the results to download
